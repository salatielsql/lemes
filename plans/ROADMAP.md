# Roadmap

## Phase 1

Creator-focused phase, to allow them to manage courses and contents using the admin dashboard.

**Goals:** 

- Creator can setup the admin account
- Creator can create a course, modules and lessons
- Creator can create and manage different types of content (text and video for now)
- API docs
- Minimal student frontend

## Phase 2

Metric-focused phase

**Goals:** 

- Creator can see the enrolled students and their metrics
- Creator can course metrics

## Phase 3

Student-focused phase

**Goals:** 

- Better student frontend
