# Lemes - A open-source Learning Management System

The main idea of this project is to create a LMS software that allows anyone to create and manage their own courses. Can use used for independent teachers, entrepreneurs, or even companies that want to create their own internal learning platform.

## What is Lemes?

- **User land** - _Where users can see the courses available and purchase them_
- **Creator Camp** - _User management, course management, reports, and platform settings_
- **API** - _RESTful API for integration with other systems_
- **Packages** (maybe) - _Modular packages, allowing you to install packages them independently_

## Features (Planned)

- User authentication and role-based access control.
- Cutomizable storefront for course listings.
- Multiple players integration (Vimeo, YouTube, Mux, Hls).
- Course creation and management.
- Some type of report: Student enrollment and progress tracking.
- RESTful API for seamless integration.

## Tech Stack (To be decided)

- **Frontend**: Vue.js
- **API**: Node.js (Express.js)
- **Database**: PostgreSQL
- **Cache**: Redis
- **File Storage**: AWS S3-like interface
- **Authentication**: JSON Web Tokens (JWT)
